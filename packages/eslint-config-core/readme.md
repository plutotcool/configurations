# ESlint core configuration

> Base eslint configuration for plutot.cool projects.

## Installation

```bash
npx install-peerdeps -P -D @plutotcool/eslint-config-core
```

In `.eslintrc`

```json
{
  "extends": "@plutotcool/eslint-config-core"
}
```

In `package.json`

```json
{
  "scripts": {
    "lint": "eslint ."
  }
}
```
