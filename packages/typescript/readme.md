# Typescript

Base typescript configs we use by default.

## Installation

```bash
yarn add -D @plutotcool/typescript
```

### Backend / Library

In your tsconfig.json

```json
{
  "extends": "@plutotcool/typescript/lib.tsconfig.json"
}
```

### Dom

In your tsconfig.json

```json
{
  "extends": "@unlikelystudio/bases-typescript/dom.tsconfig.json"
}
```
